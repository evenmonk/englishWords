REST API и админка для мобильного приложения по изучению английских слов Концепция приложения: существуют разные категории: People and things, Appearance and character, Time and dates. Они содержат темы (Human body, Face, Things), которые разделены на уровни владения языком (pre-intermediate, intermediate, upper_intermediate). В каждой теме есть набор слов.
